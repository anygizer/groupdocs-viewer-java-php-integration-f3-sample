GroupDocs.Viewer for Java integration with PHP -- PHP part
==========================================================

This integration example is based on the F3 Framework. Together with the Java application it gives a fully functional GroupDocs.Viewer installation.

> Note: Code in this repository is only one part of the integration process.

A complete guide with descriptions and other code parts is provided in the [GroupDocs documentation wiki](http://groupdocs.com/docs/display/viewerjava/How+to+Integrate+the+GroupDocs.Viewer+for+Java+Library+into+Your+PHP+Web+App).
